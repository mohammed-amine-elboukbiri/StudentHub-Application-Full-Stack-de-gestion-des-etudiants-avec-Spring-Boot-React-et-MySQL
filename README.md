# 🎓 StudentHub — Full Stack Application

> Une application web complète de gestion des étudiants, développée avec **Spring Boot** (backend) et **React** (frontend), connectée à une base de données **MySQL**.

---

## 📸 Aperçu

StudentHub permet de gérer facilement les étudiants :
- ➕ Ajouter un étudiant
- ✏️ Modifier les informations d'un étudiant
- 🗑️ Supprimer un étudiant
- 📋 Afficher la liste et le statut des étudiants

---

## 🛠️ Stack Technique

### Backend
| Technologie | Rôle |
|-------------|------|
| Java 25 | Langage principal |
| Spring Boot 3 | Framework backend |
| Spring Data JPA | Accès base de données |
| MySQL | Base de données relationnelle |
| Swagger / OpenAPI | Documentation automatique de l'API |
| JUnit 5 + Mockito | Tests unitaires |
| Maven | Gestion des dépendances |

### Frontend
| Technologie | Rôle |
|-------------|------|
| React 18 | Framework UI |
| Axios | Requêtes HTTP vers l'API |
| React Router | Navigation entre pages |
| CSS / Tailwind | Styles |
